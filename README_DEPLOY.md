# Project 1 — coins.jsl-ian.com

Pure static website. No Worker, D1, KV, secrets, or API calls.

## Deploy
1. Upload this folder to a new GitHub repository or a dedicated branch.
2. Create a Cloudflare Pages project connected to that repository.
3. Build command: leave empty.
4. Build output directory: `/` (repository root).
5. Attach custom domain `coins.jsl-ian.com`.

The `CNAME` file is included for GitHub Pages compatibility.
